Coding Project: "UFO Dataset"

UFO Sightings: Fact of Fancy? This project retrieves an API that includes all of the latest sightings of UFOs (Dataset of over 150+ UFOs sightings). The data is ported into Bootstrap and uses Javascript dynamic functions / filters to transform the data. Filters include: Date, City, State, Country, Shape.

Scope

This project intends to work on the intersection between large datasets, its API retrieve, and its front-end accessability (JavaScript Filter). The goals of this challenge are to:

01.

JavaScript Framework

Create, update, and deploy JavaScript functions to provide additional table filters. (Date, City, State, Country, Shape)

02.

Loops!

Update and deploy forEach (for loop) to loop through the filters and update them with user input.

03.

Amazing Results

Update and populate the dynamic filters and table using JavaScript and HTML.

Technologies Used

In this project, I used Javascript, CSS, HTML, Bootstrap, Chrome Developer Tool
