Coding Project: "Mapping Earthquakes"

The project visually presents the differences between the magnitudes of earthquakes all over the world for the last seven days. Using JavaScriptâ€™s D3 library, I retrieved GeoJSON data from the USGS website. (Num of Earthquakes, coordinates, magnitudes, etc.) I used the Leaflet library to plot the data on a Mapbox map through an API request and created interactivity for the earthquake data.

Scope

To complete this project, I used a URL for GeoJSON earthquake data from the USGS website and retrieved geographical coordinates and the magnitudes of earthquakes for the last seven days. Then add the data to a map.

I added a third map style as an additional tile layer. I also added tectonic plate GeoJSON data to the map to illustrate the relationship between the location and frequency of seismic activity and tectonic plates.

01.

Mapbox Map

Use d3.json() to get tectonic plate data and add the data to Mapbox Map using the L.geoJSON() layer.

02.

Transform 1/2

Style the tectonic plate LineString data to stand out on the map.

03.

Transform 2/2

Add the tectonic plate data as an overlay with the earthquake data.

04.

User Interactivity

Add a third map style to allow the user to select from three different maps.

Technologies Used

In this project, I used Javascript, CSS, HTML, GeoJSON, Mapbox API, Leaflet.
